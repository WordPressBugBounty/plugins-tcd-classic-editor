This is a classic editor extension plug-in for TCD users. It is currently offered as a beta board.

== Description ==

This plugin is a tool for those who want to use the Classic Editor more conveniently.

= Functions =

* Quick tags
* Google Maps
* Table of Contents

= Third Party Services =

This plugin relies on the following third-party services:

= Google Maps API =

- **Service URL**: [Google Maps API](https://developers.google.com/maps)
- **Terms of Use**: [Google Maps API Terms of Use](https://developers.google.com/maps/terms)
- **Privacy Policy**: [Google Privacy Policy](https://developers.google.com/maps/terms-20180207)

The Google Maps API is used to embed maps on your website. Your API key is sent to Google to authenticate the requests.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install via the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Use the "TCD Classic Editor" menu item in the WordPress admin to configure your settings.
